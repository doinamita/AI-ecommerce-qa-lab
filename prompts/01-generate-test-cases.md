# AI Prompt 01 - Generate Risk-Based Test Cases

Use this prompt with ChatGPT or another AI assistant.

```text
You are a Senior QA Lead helping me test a demo e-commerce page.

Application functionality:
- Product catalog
- Product search
- Product sorting
- Add to cart
- Remove from cart
- Coupon code QA10 gives 10% discount
- Invalid coupon should be rejected
- Checkout form requires name, email, and address
- Order confirmation appears after successful checkout

Task:
Generate risk-based test cases for this application.

Rules:
1. Include positive, negative, edge case, and accessibility scenarios.
2. For each test case include: ID, priority, type, preconditions, steps, expected result.
3. Do not invent features that are not listed above.
4. Keep expected results measurable and automation-friendly.
5. Highlight the top 5 highest-risk scenarios.
```
